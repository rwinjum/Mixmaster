🎵 MIXMASTER Instructions

DJ Playlist Organizer - Step-by-Step Guide

📋 Overview

MIXMASTER is an AI-powered DJ playlist organizer that groups songs into sets of 3 using the classic DJ "group of three" rule. It analyzes genre, era, and pace to create smooth, musically coherent transitions — and uses a local AI (Ollama) to rate each group's compatibility.

🚀 Quick Start

1Export Your Spotify Playlist

Use Exportify to export your playlist as a CSV:

  1. Go to https://exportify.app
  2. Sign in with your Spotify account
  3. Select your playlist and click Export
  4. Save the downloaded CSV file
💡 Since November 2024, Spotify removed audio features (BPM, energy, etc.) from its API. Exportify CSVs will only contain basic track info — use the Enrich metadata button after loading to fill in genres and BPM estimates.

2Upload Your CSV

  1. Click the upload zone (or drag and drop your CSV file)
  2. MIXMASTER loads all tracks and shows the song count
  3. Click 🔍 Enrich metadata to auto-detect genres and estimate BPM for each song
✅ Enrich metadata uses a built-in artist→genre database covering 150+ artists, plus Last.fm lookup for anything not in the database.

3Choose Mixing Rules & Organize

  1. Select one or more mixing rules: By Genre, By Era, By Pace, or combinations
  2. Optionally enable Avoid same artist in a row
  3. Click 🚀 Start the mix
  4. Songs are grouped into sets of 3 with AI compatibility scores

🔍 Enriching Metadata (Genre & BPM)

Because Spotify no longer provides audio features, MIXMASTER enriches missing data in two ways:

  1. Built-in artist map — 150+ artists across Classic Rock, Grunge, Alt, Metal, Hip Hop, R&B, and Pop are mapped instantly with no API key needed.
  2. Last.fm lookup — For artists not in the built-in map, MIXMASTER can query the Last.fm API (free). You need a Last.fm API key for this.

Setting Up Your Last.fm API Key

  1. Get a free API key at last.fm/api/account/create (takes ~2 minutes)
  2. Open your browser's DevTools: press F12
  3. Click the Console tab
  4. Paste this command (replace YOUR_KEY_HERE with your actual key):
    localStorage.setItem('lastfm_api_key', 'YOUR_KEY_HERE')
  5. Press Enter — the key is saved in your browser permanently
  6. Click 🔍 Enrich metadata — it will now look up any remaining songs via Last.fm
💡 The key is stored only in your browser's local storage — it never leaves your machine and is not saved in any file.

🤖 AI Compatibility Scores (Ollama)

After grouping, MIXMASTER uses a local AI model (Ollama) to rate each group 1–10 for musical compatibility. This runs entirely on your machine — no internet required.

Setting Up Ollama

  1. Download and install Ollama from ollama.com
  2. Open a terminal and pull a model (only needed once):
    ollama pull mistral
  3. Start Ollama (keep this terminal open):
    ollama serve
  4. In a second terminal, start the MIXMASTER proxy server from your project folder:
    python gemini_proxy.py
  5. Reload MIXMASTER and organize your playlist — AI scores will now appear
💡 If Ollama is already running (e.g., you see "bind: Only one usage of each socket address"), skip step 3 — it's already started.

⭐ Understanding AI Ratings

🟢 8–10 Excellent
Perfect flow. Seamless tempo, key, and energy transitions.
🟡 6–7 Good
Solid grouping with compatible characteristics.
🔴 5 or below
Algorithmic grouping only — Ollama is not running.

✏️ Editing Songs

After organizing, you can fine-tune any song:

  1. Click the ⋮ menu on any song card
  2. Choose Edit song to change genre, era, pace, BPM, or key
  3. Use Move up / Move down to reorder within a group
  4. Use Find suggestions to see replacement options from your playlist
  5. Use Remove song to drop it from the mix

Use the ▲ ▼ arrows on each group header to reorder entire groups.

🎯 Mixing Rules Explained

🎵 By Genre
Groups Rock with Rock, Pop with Pop, etc.
📅 By Era
Groups 70s together, 90s together, etc.
⚡ By Pace
Groups by High / Medium / Low energy
🎯 Genre + Era
Must match both genre AND era
🔥 Pace + Era
Must match both pace AND era
💡 You can select multiple rules at once. Songs must satisfy ALL selected rules to be grouped together.

🔧 Troubleshooting

⚠️ All songs showing "Pop" genre?
Exportify CSVs no longer include genre data (Spotify API change, Nov 2024). Click 🔍 Enrich metadata after loading your CSV to auto-detect genres.
⚠️ All compatibility scores show 5/10?
Ollama is not running. Start it with ollama serve in a terminal, then run python gemini_proxy.py from the project folder.
⚠️ "ollama serve" gives a port error?
That means Ollama is already running — you're good. Just start the proxy: python gemini_proxy.py.
💡 Song era looks wrong?
Songs from compilation albums often show the compilation's release year instead of the original. Use the ⋮ menu → Edit song to correct the era manually.

❓ FAQ

Q: Do I need an internet connection?

Only for Last.fm genre lookups. The core grouping and AI scoring (Ollama) run entirely offline on your machine.

Q: How many songs can I organize?

Any size — tested with 200+ song playlists. Larger playlists take a bit longer for AI scoring since each group is analyzed sequentially.

Q: Can I run this without Python / Ollama?

Yes — the core app is pure HTML/JS. You only need Python and Ollama for AI compatibility scores. Without them, you still get full grouping, just with algorithmic 5/10 scores.
